Калибры 2230 / 2235 — список деталей Базовый калибр 2230 (час, минута, секунда) · 2235 — с мгновенным календарём

Названия деталей — по официальному каталогу запчастей Rolex (Fournitures SAV). Зелёная пометка у номера: «общая» — деталь одинакова для обоих калибров, «2230/2235» — есть варианты по калибрам (указаны в описании), «2235» — только для календарного калибра. Фотографии деталей будут добавляться по мере съёмки.
